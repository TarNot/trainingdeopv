# Git | GitHub

## Команды для работы
 - git init (создает папку .git и начинает записывать статус версий)
 - git add . | file name (начинает отслеживать коммиты выбранных файлов)
 - git status (показывает статус файлов , учитываются ли изменения или нет, в каком разделе они находятся и статус действий)
